Kinds of relationships :
	- one to one(1:1) - one record in one table can be associated with only one record in another table.
	- one to many (1:M) - one record in table 1 can be associated with many records in another table(2nd)

	- many to many (M:M) - many record in table 1 can be associated with many records in another table(2nd)

Data Export -:


-take backup of the database
-structure (create table statement)
-data(insert into table commands)
-save db into .sql file

Export structure and data -> complete backup (both schema and data)
Export structure only - only tables structure/definition, no data
Export data only - only the table contents

Folder - SQL Backups - create backup of a db

steps :
	1. create folder - sql backup
	2. workbench - server -> data export
	3. select db
	4. select dump type - data and structure or any one
	5. select export location - to a folder containing all files or in a single file
	6. click - start export
	7. check the location for output
